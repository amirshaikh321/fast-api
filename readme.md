## 📂 Project Structure (Learning-Oriented)

```bash
fast-api/
│
├── app/
│   ├── main.py        # Entry point (FastAPI app initialization)
│   ├── routes/        # Practice endpoints & API routes
│   ├── models/        # Data models (if implemented)
│   ├── schemas/       # Pydantic schemas for validation
│   └── utils/         # Helper functions / experiments
│
├── requirements.txt
└── README.md
```
## 🚀 How to Run
```bash
git clone https://github.com/amirshaikh321/fast-api.git
cd fast-api
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
